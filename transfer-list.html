Malzeme Transfer Onayi

Stok > Transfer Listesi

Ekran Bilgisi: 18 CMTR - Confirm Material Transfer Requests

Amac: Onaylanmis malzeme taleplerinin transfer islemlerini onaylamak ve depo/konum atamak.

Pattern: LIST - Filter + Expandable Rows + Transfer Confirmation

Data Structure Alanlari (Material Request Header):

  • Material request number
  • SL user info
  • Created by user info
  • Created for user info
  • Creation date time
  • Selected week number
  • Selected weeks first day, date
  • Description for the request
  • Requesting department
  • Delivery Location 1-5 id
  • Approval finish date time

Data Structure Alanlari (Material Request Item):

  • Material number
  • Requested Quantity
  • Unit of Measure
  • Priority
  • Stock record Id
  • Assigned storage/location/bin
  • Selected to transfer
  • Delivered quantity
  • Material delivery finalized
  • Rollback initiated check

Veritabani Islemleri

API Endpoints:

  • GET /api/material-requests/approved - Onaylanmis talepleri listele (Status = 'Approved', Transfer Status = 'Pending')
  • POST /api/material-transfers/confirm - Transfer onayi yarat
  • PUT /api/material-requests/{id}/assign-warehouse - Depo/lokasyon/bin atama
  • GET /api/material-requests/print - Transfer listesi yazdirma

Tablolar:

  • MAT_MaterialRequest - Malzeme talep basliklari (Material Request number, SL user info, Created by, Created for, Creation date time, Selected week number, Selected week's first day, Description, Requesting department, Delivery Location 1-5 id, Approval finish date time)
  • MAT_MaterialRequestItem - Talep kalemleri (Material number, Requested Quantity, Approved quantity, Transfer quantity)
  • INV_WarehouseAssignment - Depo/lokasyon/bin atamalari (Warehouse Assignment)
  • MAT_TransferConfirmation - Transfer onay kayitlari

Siralama: ApprovalFinishDate ASC, CreationDate DESC
